Features

  • Login

    The login system works if you type in an existing user's username and password. To find that, click on the tour tab to see the list of users. Within login, you can also create a new user. Filling in the forms will create a new user.

  • Tour

    Displays a list of users. The password will not be shown in the final version, but it's there to show that how the login system works.

  • Database

    The product systems allows the user to create, delete, and update items in the database. The list of items are also displayed within the same page. Filling out the forms will generate the product.

  • About

    A page that describes what our project does and what our goals are.
